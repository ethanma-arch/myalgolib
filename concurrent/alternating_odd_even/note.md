# 交替打印奇偶数

两个 goroutine，一个只打印奇数（1,3,5,...），一个只打印偶数（2,4,6,...），要求按顺序输出 1,2,3,4,...,N。

## 思路

用两个无缓冲 channel 做「令牌」：  
- 打印奇数前要从 `oddCh` 取到信号；打印完后给 `evenCh` 发信号，让偶数 goroutine 打印。  
- 打印偶数前要从 `evenCh` 取到信号；打印完后给 `oddCh` 发信号，让奇数 goroutine 打印。  
主 goroutine 先向 `oddCh` 发一个信号启动流程。

## 输入输出（ACM 格式）

- **输入**：一行一个整数 N（1～N 要按顺序交替打印）。
- **输出**：每行一个数，顺序为 1,2,3,...,N。

## 运行

```bash
go run main.go < input.txt
```

或评测时由系统从 stdin 喂入 N。
