# 3. 无重复字符的最长子串

## 题意

在字符串中找一个连续子串，子串里没有重复字符，求其最大长度。

## 思路：滑动窗口 + 上次出现位置

- 用 **双指针** 表示当前子串 `[l, r]`，右指针 `r` 从左到右扫。
- 用 **lastPos[char]** 记录每个字符**上一次出现**的下标（-1 表示未出现过）。
- 当 `s[r]` 在 `lastPos` 里已经出现过且位置 ≥ `l` 时，说明当前窗口里重复了，把左边界移到**上一次该字符的下一个位置**：`l = max(l, lastPos[s[r]] + 1)`，这样窗口内 again 无重复。
- 每步更新 `lastPos[s[r]] = r`，并用 `r - l + 1` 更新答案。

## 要点

- 左边界 `l` 只会向右移动，不会回退。
- 用数组 `lastPos[128]`（或 map）即可，字符集有限时数组更简单。

## 复杂度

- 时间 O(n)，空间 O(字符集大小)。
